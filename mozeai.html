What is MozeAI?

MozeAI is a next-generation AI assistant designed to simplify how users interact with information. It integrates multiple advanced capabilities into a single platform, allowing users to search, analyze, and generate content seamlessly. Built using Groq's LLM API and Streamlit, MozeAI delivers accurate, context-aware, and up-to-date responses.

📄 Document Intelligence
