# MozeAI - Intelligent AI Assistant

[![Streamlit App](https://img.shields.io/badge/🚀-Live_Demo-blue)](https://moze12432appio-a65t2vjq4b28fakzt6gzbf.streamlit.app)

## Features

- 🔍 **Real-time Web Search** - Get current information instantly
- 📄 **File Analysis** - Upload and analyze PDF, DOCX, TXT, CSV, JSON
- 🎨 **Image Generation** - Create and edit images with AI
- 💻 **Coding Assistant** - Get help with Python, JavaScript, HTML, CSS
- 🧮 **Smart Tools** - Calculator, news headlines, date/time

## Tech Stack

- Streamlit for UI
- Groq LLM models and fall back for reliability 
- Pollinations.ai for image generation
- BeautifulSoup for web scraping
